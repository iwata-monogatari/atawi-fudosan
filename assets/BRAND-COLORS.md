# ATAWI FUDOSAN ブランドカラー定義（看板準拠）

富士ヶ丘サービスの屋外看板「不動産売却／1円でも高く／TEL 0538-31-3308」の配色を、
サイト全体の色の基準にする。**空色ブルー・看板イエロー・白・濃紺の4色**だけで組む。
茶・金・緑・オレンジといった旧アース系の色はサイト内から全廃した。

正の定義元は `assets/site-header.css` の `:root`（全ページで読み込み済み）。
各ページの `:root` は同じ値をローカル名で持つ。**色を変えるときは必ずここから直す。**

## パレット

| 役割 | 値 | 使いどころ |
|---|---|---|
| 看板ブルー | `#2E9BD4` | ヒーロー上部・帯・図版の面。**この上に白文字は置かない**（3.1:1） |
| ブルー（濃） | `#16709F` | 白地に置くリンク・見出し・ボタン地。白文字OK（5.5:1） |
| ブルー（最濃） | `#0E5C8A` | セクション見出し・グラデーション終端 |
| 濃紺 | `#0B3C5D` | 上部帯の地色、黄色い面の上の文字、最も強い見出し |
| ペールブルー | `#A9DBF3` | 図版の面 |
| 淡水色 | `#E9F5FC` / `#D3E9F7` | セクション地・タグ |
| 地色 | `#F6FAFD` / `#F2F9FE` | ページ地 |
| 看板イエロー | `#FFE100` | **青地・濃紺地の上のみ**。帯の中の強調文字 |
| イエロー（濃） | `#F2B705` | 図版の面・アイコン・CTAの地（文字は濃紺） |
| アンバー（文字用） | `#8F6200` | 白地に黄系の文字を置くときだけ（5.4:1） |
| 本文 | `#14344A` | |
| 補助文字 | `#5B7488` | 白地で4.9:1 |
| 罫線 | `#CDE2F0` | |

## 使用ルール

1. **黄色は面、青は文字。** 看板と同じく、黄色は必ず濃紺か白の輪郭・地と組む。
   黄色い面に白文字を置くと 2.1:1 で読めないため、黄色い面の文字は必ず `#0B3C5D`。
2. **白地に黄色い文字を置かない。** どうしても必要なら `#8F6200`。
3. **`#2E9BD4` は面の色。** 白文字を載せるボタン・帯は `#16709F` 以上の濃さにする。
4. 例外は外部ブランド色のみ：LINE `#06C755`、警告 `#C0392B`。

## 検証

全ページで文字とその背景のコントラスト比を計測し、WCAG AA（通常 4.5:1 / 大文字 3:1）を
満たすことを確認済み。唯一の未達は LINE 公式ブランド色のボタン（`#06C755` + 白文字＝2.26:1）で、
これはブランド指定のため据え置き。
